## Documentation

- [API reference (master branch)](https://github.com/tjardoo/eloquent-rs)
- [API reference (docs.rs)](https://docs.rs/eloquent/latest/eloquent)

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
eloquent = "0.1.1"
```
